# schemadoc (java swing)
DB 명세서를 엑셀로 출력하기 위한 프로그램

### 지원 DB
```
- MySQL
- Oracle
- MariaDB
- PostgreSql
- MSSQL
- Cubrid
- Tibero
```
### 사용방법
```
1. https://github.com/hunecenter94/schemadoc/tree/main/jar/dbschema.jar 다운로드
2. https://github.com/hunecenter94/schemadoc/tree/main/bat/schema.bat 다운로드
3. bat파일에서 메모장 또는 에디터 편집파일로 java 명령어 호출 부분 경로 변경
   ㄴ (예시)java -jar jar/dbschema.jar 
```
